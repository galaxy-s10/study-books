# 简介

HTML 特性是不区分大小写的。所以，当使用的不是字符串模版，camelCased (驼峰式) 命名的 prop 需要转换为相对应的 kebab-case (短横线隔开式) 命名： 如果你使用字符串模版，则没有这些限制。”

# camelCased（驼峰式/小驼峰）

# PascalCase（帕斯卡命名法/大驼峰）

> 与骆驼命名法类似。只不过骆驼命名法是首字母小写，而帕斯卡命名法是首字母大写

# kebab-case（短横线命名）
