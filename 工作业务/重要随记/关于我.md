## 技术栈

> 本站前台前期使用vue全家桶进行开发，后面改用了Nuxt.js进行重构。后台前期使用vue-admin-template，后面使用Vue全家桶以及iView搭建后台

- **Client:**  ~~vue全家桶~~  Nuxt.js + ElementUI
- **Admin:**  ~~vue-admin-template~~  Vue全家桶 + iView
- **Serve:** Node + Express + Sequelize + Mysql
